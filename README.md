A full-stack **CRUD (Create, Read, Update, Delete)** web application built using **Node.js, Express.js, and EJS**, following a structured **MVC architecture**.

---

## 📖 What is this Project?

This is a **Contact Management System** where users can:

- ➕ Add new contacts  
- 📄 View all contacts  
- ✏️ Update contact details  
- ❌ Delete contacts  

👉 It represents how real-world applications manage and manipulate data.

---

## 🎯 Purpose of the Project

- Understand CRUD operations in backend development  
- Learn Express routing & middleware  
- Work with EJS for dynamic UI rendering  
- Follow MVC (Model-View-Controller) architecture  
- Build a strong full-stack foundation
